AniLet.js

v1.1.0

Class-driven animated letters in vanilla JS. Add the class you want, and the text animates when it enters the viewport. Control timing via data-delay and data-duration.

Install / Use

Include script 
<script src="aniLet.min.js"></script>
Basic usage 
<div class="aniLet aniLet__fadeIn">Hello AniLet</div>

Place aniLet.min.js alongside this page (in docs/ for GitHub Pages). Options: data-delay (ms between letters, default 50) and data-duration (total ms, default 1000).
