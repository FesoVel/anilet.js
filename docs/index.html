AniLet.js

v1.1.0

AniLet.js is a super lightweight JavaScript library to apply animated reveal of each letter individually when they appear in the viewport.

Super fast, only 2.67 KB minified. No dependencies, use it anywhere.

Features

  • Animate text letter-by-letter
  • 11 Built-in animations
  • Trigger animations on scroll
  • Pure vanilla JavaScript

Installation using NPM

Install the package 
npm install ani-let

Then import the built file (side-effect import initializes the library): 
import 'ani-let/dist/aniLet.min.js';

Installation using a CDN (no build step)

Install the package 
npm install ani-let

Then import the built file (side-effect import initializes the library): 
            <!-- jsDelivr -->
<script src="https://cdn.jsdelivr.net/npm/ani-let/dist/aniLet.min.js"></script>
<!-- or unpkg -->
<script src="https://unpkg.com/ani-let/dist/aniLet.min.js"></script>

Installation using a local file

Copy 
dist/aniLet.min.js
into your project and include it: 
<script src="/path/to/aniLet.min.js"></script>

Usage

Add 
aniLet
and one animation class to any element. Animations trigger when the element enters the viewport. 
<script src="aniLet.min.js"></script>

<div class="aniLet aniLet__fadeIn">Hello AniLet</div>
<div class="aniLet aniLet__slideUp">Slide Up</div>
Optional timing attributes: 
  • data-delay
    : delay between letters in ms (default:50)
  • data-duration
    : total animation duration in ms (default:1000)
<div class="aniLet aniLet__zoomIn" data-delay="80" data-duration="800">Zoom In Example</div>
